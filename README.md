# FlashChat

IOS app. Internet based messaging app similar to WhatsApp.

Concepts:

* uitableview
* cocoapods 
* navigation-controller 
* delegates 
* segues 
* firebase-database 
* firebase-authentication 
* firebase-firestore 
* xib-files 
* applifecycle
